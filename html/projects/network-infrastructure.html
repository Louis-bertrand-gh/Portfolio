Description

Contexte du projet

Dans le cadre de ma formation BTS SIO option SISR, En groupe de 3 étudiants, nous avons conçu et déployé une infrastructure réseau complète et concrète dans un labo de test.

L'objectif était de simuler un réseau d'entreprise avec segmentation, Amélioration en continu , ajout de differents services, et mise en place de mesures de sécurité pour protéger les données et les utilisateurs.

Ce projet m'a permis de mettre en pratique l'ensemble des compétences techniques acquises en formation : virtualisation, configuration réseau, administration système et cybersécurité.

Objectifs

Ce qui a été réalisé

  • Déploiement d'un hyperviseur Proxmox comme socle de virtualisation pour heberger nos machines virtuelles
  • Sécurisation de l'infrastructure (VLAN, ACL, politiques de groupe, parefeu Opensense et StormShield)
  • Installation et configuration des services (DHCP, DNS, AD, Apache, HMail, Graylog, BDD, Ansible, ReverseProxy, Docker, Wordpress )
  • Rédaction d'une documentation technique complète
Schéma d'architecture réseau

Architecture

Topologie réseau

L'infrastructure est structurée autour d'un cœur de réseau pfSense qui assure le routage inter-VLAN et le filtrage des flux entre les différentes zones de sécurité.

  • VLAN 10 — Administration : serveurs, contrôleurs de domaine
  • VLAN 20 — Utilisateurs : postes de travail, accès Internet filtré
  • VLAN 30 — DMZ : services exposés (Apache, FTP)
  • VLAN 99 — Management : accès administration des équipements

Technologies utilisées

Stack technique

Proxmox VE OpnSense Windows Server 2022 Active Directory DHCP / DNS Cisco Apache Ansible Docker Graylog WordPress MySQL
